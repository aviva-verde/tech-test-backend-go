# Aviva Zero - Full Stack Tech Test (Go)

![Preview of Frontend](preview.png)

## Prerequisites

* Go 1.12
* Access to the internet
* Suitable development environment

## Getting Started

* Build: 
```shell
make build
```
* Run: 
```shell
make run
```
* Test: 
```shell
make test
```
* View: 
```shell
make run
```
then navigate to http://localhost:8080
