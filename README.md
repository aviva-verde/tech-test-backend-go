# Infinity Works - Full Stack Tech Test (Go)

![Preview of Frontend](preview.png)

## Prerequisites

* Go 1.12
* Access to the internet
* Suitable development environment

## Getting Started

* Build: `make build`
* Run: `make run`
* Test: `make test`
* View: `make run` then navigate to http://localhost:8080
